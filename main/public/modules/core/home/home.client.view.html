PCE Virtual Laboratory

Playing with Crop Simulation Models was never easier!

The main goal of this page is to provide a Virtual Laboratory for testing crop simulation models and providing the test-bed for future implementation of decision support systems (DSS) for scientists and farmers.

Currently we inthe process of integrating Python Crop Simulation Environment - PCE, which is a Python package for building crop simulation models and in particular the crop models developed in Wageningen (Netherlands).

This WebApp has been developed by Dr Jose Jimenez-Berni, from the IAS/CSIC in Cordoba, Spain. For more information please contact!.
